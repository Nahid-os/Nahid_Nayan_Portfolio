# Md Nahid Husen Nayan — Portfolio

A modern, animated personal portfolio built with **Next.js 15**, **React 19**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Sleek dark theme with animated gradient accents and glassmorphism cards
- Typewriter hero, scroll progress bar, and scroll-reveal animations
- Sections: Hero, About + Education, Experience timeline, Projects, Skills, Contact
- Fully responsive with a mobile nav menu
- Optimised fonts via `next/font` and good SEO metadata

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
