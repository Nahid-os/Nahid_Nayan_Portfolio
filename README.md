# Md Nahid Husen Nayan — Portfolio

A modern, animated personal portfolio built with **Next.js 15**, **React 19**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Sleek dark theme with animated gradient accents and glassmorphism cards
- Typewriter hero, scroll progress bar, and scroll-reveal animations
- Sections: Hero, About + Education, Experience timeline, Projects, Skills, Contact
- Fully responsive with a mobile nav menu
- Optimised fonts via `next/font` and good SEO metadata

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new), import the repo.
3. Vercel auto-detects Next.js — just click **Deploy**. No extra config needed.

## Editing your content

All content lives in [`lib/data.ts`](lib/data.ts) — update your summary, experience,
projects, and skills there and the whole site updates.
